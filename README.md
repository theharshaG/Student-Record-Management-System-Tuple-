# Student Record Management System (Python)

## Project Overview
This is a simple **menu-driven Python program** that manages student records using a list of tuples.  
Each student has a roll number, name, and marks in three subjects.

The project is designed for beginners to understand:
- Tuples
- Lists
- Loops
- Conditional statements
- User interaction
- 
## Features
- Display all student records in a table format
- Search a student using roll number
- Calculate overall average marks
- Simple and easy-to-read menu system

## Concepts Used
- Python Lists
- Tuples
- Tuple Unpacking
- For Loops
- If–Else Conditions
